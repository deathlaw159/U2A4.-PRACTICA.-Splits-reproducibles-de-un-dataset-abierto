# U2A4.-PRACTICA.-Splits-reproducibles-de-un-dataset-abierto

Este proyecto utiliza el dataset Iris dataset para dividir los datos en dos conjuntos: entrenamiento y prueba, y guardarlos en archivos .csv. El objetivo es mostrar de forma sencilla cómo preparar datos para su uso en modelos de inteligencia artificial.

Al ejecutar el código, se generan dos archivos: train.csv, que contiene los datos para entrenar el modelo, y test.csv, que contiene los datos para evaluarlo. Para poder ejecutar el proyecto, es necesario tener instaladas las librerías pandas, matplotlib y scikit-learn.

El uso es muy simple: solo necesitas ejecutar el script en Python y automáticamente se crearán los archivos train.csv y test.csv. Después, puedes ejecutar los códigos de gráficas para visualizar cómo están distribuidos los datos.

La partición del dataset se realiza usando una proporción de 80% para entrenamiento y 20% para prueba, con una semilla fija de 42 para asegurar que los resultados sean siempre los mismos. La división se hace de manera aleatoria para evitar sesgos en los datos.

En resumen, este proyecto muestra de forma práctica cómo dividir un dataset y visualizarlo, lo cual es un paso fundamental en cualquier proyecto de inteligencia artificial.
